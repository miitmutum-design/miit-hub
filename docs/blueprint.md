# **App Name**: Local Hub

## Core Features:

- User Profile Creation: Users can create and manage their profiles, including saving preferences.
- Browse Local Listings: Browse list of businesses and service providers near the user.
- Location Based Search: Filter displayed locations based on proximity.
- Offer Feed: Merchants create offer and events to which users can subscribe.
- Business Review Analysis: Analyze reviews of the merchants, creating an aggregate score to rank them in order of value. Use AI tool to find general satisfaction with business
- Direct Messaging: Secure channel for merchants to engage with local customer base

## Style Guidelines:

- Primary color: Dark saturated green (#54B435) for a vibrant yet natural feel, referencing local businesses and growth.
- Background color: Very dark desaturated gray-green (#1A1A18), to make the vivid primary color stand out while still maintaining consistency
- Accent color: Deep turquoise (#449775) for interactive elements and calls to action.
- Body and headline font: 'PT Sans' (sans-serif), which provides modern legibility and subtle personality.
- Use clear, outlined icons for easy navigation.
- Cards with rounded corners for business listings.
